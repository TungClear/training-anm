# Bank Account Backend

## How to start

- Copy file .env from .env.example:

- Install packages:
```sh
pip install -r requirements.txt
```

- Pre-populate users for authentication for first time:
```sh
python app.py seed_db
```

- Run app:
```sh
python app.py runserver
```

## Running Test

```sh
sh run_test.sh
```
